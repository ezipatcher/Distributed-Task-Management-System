# Project Description

This repository contains the implementation and documentation of Distributed Systems Thinking concepts including Loose Coupling, Service Communication, and Failure Isolation using modular architecture.

---

# Technology Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python FastAPI / Node.js
* Database: PostgreSQL / MySQL
* Communication: REST APIs
* Containerization: Docker
* Message Broker: Redis
